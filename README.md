# 🌟 **KeenKeeper — Stay Connected, Stay Close**

## 🚀 **Project Overview**

**KeenKeeper** is a modern friendship management web application designed to help users stay connected with the people who matter most. It allows users to track interactions, set communication goals, and maintain meaningful relationships effectively.

---

## 🛠️ **Technologies Used**

* ⚛️ **Frontend:** React.js (ES6+)
* 🔗 **Routing:** React Router DOM
* 🎨 **Styling:** Tailwind CSS & DaisyUI
* 📊 **Data Visualization:** Recharts
* 🎯 **Icons:** Lucide React / FontAwesome
* 🔔 **Notifications:** React Toastify

---
